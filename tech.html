HTML Basics

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

HTML uses a system of tags to define the structure of web content.

For example, <h1>Header</h1> is used for the main header of a page.

HTML5 introduced new elements like <article> and <section>.

<!DOCTYPE html> <html> <head>
  • <title> tag defines the document's title.
  • <meta> tag provides metadata about the document.
  • <body> contains the visible content of the document.
CSS Styling

CSS stands for Cascading Style Sheets.

It is used for styling the HTML structure.

CSS can be applied inline, in the head section, or through an external stylesheet.

body { background-color: lightblue; } h1 { font-size: 36px; }
  • Selectors are used to target elements.
  • Properties define the styles to apply.
  • CSS supports responsive design through media queries.
JavaScript Basics

JavaScript is a programming language used to create dynamic web content.

It allows interaction with users and manipulation of the DOM.

let x = 10; console.log(x);
  • Variables can be declared using let, const, or var.
  • Functions can be defined using function keyword.
  • Events such as clicks or keypresses can trigger JavaScript functions.
Web Development

Web development involves building websites using various technologies.

It includes front-end (HTML, CSS, JavaScript) and back-end (server-side) development.

const express = require('express'); app.listen(3000, () => { console.log('Server running'); });
  • Front-end development deals with user interfaces.
  • Back-end development involves server-side scripting.
  • APIs allow communication between front-end and back-end.
